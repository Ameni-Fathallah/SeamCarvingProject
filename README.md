# Seam Carving 
# Description
Implementation of the Seam Carving algorithm to resize images by removing the least important pixels, while preserving the essential visual content.


# Installation
```bash
# 1. Clone the repository
git clone <your-repo>
cd SeamCarvingProject

# 2. Install dependencies
pip install -r requirements.txt

# 3. Prepare the input image # Place an image named 'image.jpg' in 'images/input/'
